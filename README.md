# Mandar Glassmorphism Portfolio

A modern single-page portfolio website with:
- Glassmorphism UI
- Wallpaper-style background image
- Profile image section
- Project image cards
- Hover animations
- Floating mini cards
- Music player
- Responsive mobile layout

## Replace these files

Put your own files inside the `assets` folder using the same names:

- `background.jpg` = your website wallpaper/photo
- `profile.jpg` = your profile photo
- `project1.jpg` = AWS/EKS project screenshot
- `project2.jpg` = Jenkins/CI-CD screenshot
- `project3.jpg` = monitoring/project screenshot
- `song.mp3` = your song/music file

## Run locally

Just open:

```bash
index.html
```

## Deploy on GitHub Pages

```bash
git init
git add .
git commit -m "add glass portfolio website"
git branch -M main
git remote add origin git@github.com:maddy033/portfolio.git
git push -u origin main
```

Then go to GitHub repository:
Settings → Pages → Deploy from branch → main → root.
